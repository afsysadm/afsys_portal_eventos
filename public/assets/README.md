# Assets do tenant

Coloque aqui a logo (brasão) de cada sindicato. O caminho é referenciado em
`src/config/tenants.ts` (campo `logo`) e servido em `/assets/...`.

## Pendente

- `logo-hoteleirorp.webp` — brasão circular do Sindicato dos Hoteleiros de
  Ribeirão Preto (fundo transparente/branco, ~256×256). Enquanto o arquivo não
  estiver aqui, o header/rodapé exibem um monograma com as iniciais sobre a cor
  da marca (fallback automático em `TenantLogo`).
